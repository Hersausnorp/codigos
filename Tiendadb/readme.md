# Sistema de Gestión de Cuentas – Tienda de Barrio

Aplicación de consola desarrollada en **Python** para gestionar clientes y cuentas fiadas de una tienda, utilizando **MySQL** como base de datos y **Docker Compose** para el entorno de ejecución.

---

## 📌 Funcionalidades

- Crear clientes
- Listar clientes
- Abrir cuentas (agregar productos fiados)
- Ver cuentas activas
- Pagar cuentas (elimina la deuda, mantiene el cliente)
- Persistencia de datos en MySQL

---

## 🛠️ Tecnologías utilizadas

- Python 3.11
- MySQL 8.0
- Docker
- Docker Compose
